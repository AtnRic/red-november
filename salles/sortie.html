Merci d'avoir joué !

Adaptation de l'histoire du jeu de plateau Novembre Rouge.
Réalisé par Antoine Richard et Adrien Verhaeghe.











Solution - Salle 1

L'ordre des boutons est donné par la traduction en bas de l'écran.
Après avoir cliqué sur les différents boutons, il suffit d'entrer dans l'ordre les différents numéros obtenus.
Code : 35176






Solution - Salle 2

Le but de cette salle est d'aligner les fils afin qu'ils soient de la même couleur.
Après avoir trouvé la combinaison de couleur, il suffit d'entrer dans l'ordre le résultat de chaque somme des numéros apparents entre les fils.
Combinaison de couleur : Jaune, Rouge, Bleu.
Code : 7915
Résultat des additions : (3 + 4)(4 + 5)(7 + 8)





Solution - Salle 3

Il s'agit d'une énigme audio. Il faut traduire le code morse donné par l'audio du magnétophone.
Chaque numéro a sa traduction en bas de la page.
Code : 9265






Solution - Salle 4

Il suffit de couper le fil bleu.
Les autres éléments sont factices.






Solution - Salle 5

C'est un code à trouver.
Chaque chiffre du code est donné dans chaque salle (disposé en haut à droite).
Code : 47134
